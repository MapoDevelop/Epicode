# 📘 README — Python, AI & Machine Learning (Epicode)

## 🎯 Obiettivo della Repository

Questa repository raccoglie tutti gli **esercizi, progetti pratici e materiali di studio** sviluppati durante il percorso _Python, AI & Machine Learning On‑Demand_ di Epicode.

Il focus è sulla costruzione progressiva delle competenze fondamentali per lavorare con:

- **Python**
- **Data Analysis & Data Cleaning**
- **Machine Learning classico**
- **Deep Learning con TensorFlow/Keras**
- **Computer Vision & NLP**
- **AI Generativa e LLM**

---

## 📂 Struttura della Repository

- **01_python_basics** — esercizi introduttivi su sintassi, funzioni, strutture dati.
- **02_data_analysis** — notebook su pulizia dati, pandas, visualizzazioni.
- **03_machine_learning** — modelli predittivi, regressione, classificazione, metriche.
- **04_deep_learning** — reti neurali, TensorFlow, Keras, training e valutazione.
- **05_computer_vision** — CNN, image classification, dataset augmentation.
- **06_nlp** — tokenizzazione, modelli sequenziali, sentiment analysis.
- **07_ai_generativa** — GAN, LLM, prompt engineering, applicazioni end‑to‑end.
- **final_project** — progetto finale su un caso reale.

## 📌 Come Eseguire gli Esercizi

1. Clona la repo
   ```bash
   git clone https://github.com/<username>/<repo>
   ```
2. Crea un ambiente virtuale
   ```bash
   python -m venv venv
   source venv/bin/activate  # macOS/Linux
   venv\Scripts\activate     # Windows
   ```
3. Installa le dipendenze
   ```bash
   pip install -r requirements.txt
   ```
4. Avvia Jupyter
   ```bash
   jupyter notebook
   ```
